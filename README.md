# Dotfiles

* OS - Arch
* Display Manager - Ly
* Window Manager - Hyprland
* Browser - Firefox
* Terminal - Kitty
* Shell - Zsh
* Editor - Nvim
* Launcher - Fuzzel
* Theme - Tokyo Night
* Bar - Waybar
* Font - JetBrainsMono Nerd Font
